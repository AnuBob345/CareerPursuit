# CareerPursuit

A Python application for career information and pursuit, built with CustomTkinter. Features include user login and a main screen displaying career-related information.

## Installation

1. Ensure you have Python 3.10 or higher installed.
2. Install Poetry if not already installed: `pip install poetry`
3. Clone the repository: `git clone https://github.com/AnuBob345/CareerPursuit.git`
4. Navigate to the project directory: `cd CareerPursuit`
5. Install dependencies: `poetry install`

## Usage

Run the application with: `poetry run python main.py`

This will launch the login screen. After logging in, access the main screen with career information.

## Requirements

- Python >= 3.10.0, < 3.12
- CustomTkinter ^5.2.2
